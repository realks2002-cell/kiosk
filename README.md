# 국풍국어학원-신갈관 키오스크

중·고등 학원을 위한 무인 출결·자습실 관리 키오스크 데모. 학생이 직접 등하원을 체크인하고, 자습실 좌석·수강·교재·면담을 셀프로 신청하는 태블릿 키오스크입니다.

## 기능

- **등원/하원 체크인** — QR 또는 PIN 4자리, 완료 시 학부모 알림톡 미리보기
- **자습실 자리 예약** — 1자습실(20석·4×5) / 2자습실(16석·4×4) 좌석맵, 상태별 색 구분, 이용 시간 지정
- **수강신청** — 고등부 국어 강좌(강사·요일/시간·잔여석) 선택
- **교재신청** — 교재 복수 선택 + 합계 계산
- **면담신청** — 선생님·날짜·시간 선택 예약
- **학원소식 / 공지** — 소식·공지·일정 상시 노출
- 작업 완료 후 5초 카운트다운 자동 홈 복귀

## 스택

- 순수 **HTML + CSS + Vanilla JavaScript** (빌드 없음, 단일 파일 SPA)
- 폰트 **Pretendard**, 아이콘 **Lucide** (CDN)
- 태블릿 세로(800×1280) 포트레이트 최적화

## 디자인

- 블루 그라데이션 캔버스 + 흰 카드 + 블루 브랜드(`#2563EB`) + 레드 포인트(`#EF4444`)
- Radix 기반 컬러 토큰, 4/8 그리드 여백, 카드 라운드 5px, WCAG AA 대비 검증

## 실행

```bash
python3 -m http.server 8731
# http://localhost:8731/kiosk.html
```

## 파일

- `color-palette.html` — 컬러 팔레트 시트
